Docker Desktop Extension Preview

IoTSharp in one install

这个扩展会在 Docker Desktop 里自动拉起一个 SQLite 版 IoTSharp 体验环境，适合首次试用、演示和文档截图。

打开 IoTSharp 打开 Installer 阅读文档
Web UI http://localhost:2927
MQTT localhost:1883 / 8883
CoAP localhost:5683 / 5684
Modbus TCP localhost:1502
Database Embedded SQLite

第一次打开

首次访问通常会进入安装引导页，请在浏览器里完成管理员账号与租户的初始化。

  • 默认环境：`Sqlite`
  • 持久化卷：`iotsharp-dd-data`、`iotsharp-dd-security`
  • 扩展卸载前请先导出重要数据

适合这版的目标

  • 给新用户一键体验 IoTSharp 核心界面
  • 给演示、培训、文档截图提供固定环境
  • 给后续 PostgreSQL / RabbitMQ 扩展版提供分发骨架

下一轮建议

  • 增加扩展内状态探测与日志查看
  • 支持在扩展 UI 里切换 SQLite / PostgreSQL 体验模板
  • 把运行镜像切到 GHCR 与 Docker Hub 双发布通道

仓库与发布

首版扩展镜像会优先服务 Docker Desktop 安装与本地内测。后续可以把同一镜像推送到 Docker Hub 和 GitHub Container Registry， 并在 GitHub Actions 的 Tag 发布阶段自动执行 `docker extension validate` 与镜像推送。
