# 产品概览

IoTSharp 是一个面向设备接入、数字孪生、规则处理与运维交付的开源物联网平台。

## 核心能力

- 多协议接入：支持 HTTP、MQTT、CoAP 等协议接入设备或网关。
- 数字孪生：围绕设备、网关、产品、资产建立统一的数据与控制模型。
- 规则链：通过规则链设计器、脚本与执行器完成数据清洗、告警、通知与业务动作。
- 遥测存储：支持关系库与时序数据库存储遥测数据。
- 多租户：系统管理员、租户管理员、客户管理员和普通用户拥有不同菜单范围。
- 运维交付：支持 Docker、Windows 服务、Linux 服务、NuGet、GitHub Release 与容器镜像发布。

## 适合的使用场景

- 设备监控平台
- 工业网关与边缘接入
- 产品化设备管理后台
- 规则驱动的告警与联动
- 多租户 IoT SaaS 平台

## 产品入口

用户在浏览器中通常会遇到三个关键入口：

- `/installer`：首次安装后的初始化入口。
- `/login`：正常登录入口。
- `/dashboard`：登录后的平台首页。

## 角色差异

IoTSharp 的菜单受角色影响：

- `SystemAdmin`：可见证书管理、租户列表。
- `TenantAdmin`：可见客户列表。
- `CustomerAdmin`：可见用户列表。
- `NormalUser`：可见设备、告警、规则链、场景、产品、资产等日常业务菜单。
