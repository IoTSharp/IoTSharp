# Windows 服务部署

IoTSharp 主程序已经支持作为 Windows 服务运行。

## 当前交付方式

Windows 环境建议使用 MSI 安装包：

- 自动安装程序文件
- 自动注册 `IoTSharp` 服务
- 默认设置为自动启动

## 服务模式的关键点

为了避免服务模式下相对路径漂移，当前主程序已经在启动时把工作目录固定到可执行目录。

这对以下内容很重要：

- Sqlite 数据文件
- 证书文件
- 本地安全目录

## 安装后的建议检查

1. 在“服务”中确认 `IoTSharp` 已注册。
2. 确认服务状态为“正在运行”。
3. 用浏览器访问系统地址。
4. 如果首次安装，进入 Installer 页面继续初始化。

## IoTSharp.Agent

仓库中新增了 `IoTSharp.Agent` 跨平台托盘项目，适合后续与 Windows 安装体验配套：

- 托盘显示状态入口
- 菜单可直接在浏览器中打开 IoTSharp
- 后续可继续补自动启动与安装集成
