# 产品与资产

产品与资产分别承担“标准化定义”和“业务对象承载”两类职责。

## 产品管理

菜单入口：

- 产品管理 > 产品列表

从当前前端实现看，产品页包含：

- 产品列表
- 产品表单
- 数据字典
- 数据映射设计
- 产品属性配置
- 批量创建设备

适合场景：

- 统一设备型号定义
- 管理默认属性与接入参数
- 量产设备时减少重复配置

## 资产管理

菜单入口：

- 资产管理 > 资产列表

资产页对应前端能力包括：

- 资产列表
- 资产详情
- 资产设计器

适合场景：

- 组织设备归属关系
- 表达空间、站点、产线或园区层级
- 从业务视角聚合设备与规则

## 推荐使用方式

- 先用产品统一建模
- 再用设备完成实例化
- 最后用资产承载业务场景和层级关系
