# 系统管理

系统管理菜单会随角色变化而变化。

## 角色与菜单对应

### SystemAdmin

- 证书管理
- 租户列表

### TenantAdmin

- 客户列表

### CustomerAdmin

- 用户列表

## 证书管理

证书管理适用于：

- 设备证书体系维护
- MQTT TLS / X509 相关安全接入
- 平台证书与信任链管理

## 租户、客户、用户

IoTSharp 的组织结构通常可理解为：

- 租户：平台级业务主体
- 客户：租户下的业务客户或组织
- 用户：实际登录人员

## 管理建议

- 先规划租户边界，再建立客户层级。
- 权限尽量按角色发放，不建议长期使用超大权限账号。
- 证书和用户管理建议纳入日常审计流程。
