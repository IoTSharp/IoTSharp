# 设备与网关

设备和网关是 IoTSharp 最核心的业务对象之一，对应菜单主要集中在“数字孪生”区域。

## 菜单入口

- 数字孪生 > 设备管理

## 设备管理包含什么

从当前前端结构看，设备侧功能包括：

- 设备列表
- 设备详情
- 设备属性编辑
- 最新遥测查看
- 历史遥测查看
- 规则链绑定
- 证书下载
- 设备备份

## 网关相关页面

虽然网关设计器是隐藏页面，但代码中已经有明确入口：

- `gatewaydesigner.vue`
- `gatewaylist.vue`
- Modbus / OPC UA 相关设计面板

这意味着 IoTSharp 不只是单设备管理，也支持通过网关模型组织现场接入。

## 推荐使用流程

1. 先建立产品模型。
2. 创建设备或网关。
3. 配置属性、凭证和接入参数。
4. 验证设备上线与遥测写入。
5. 绑定规则链。

## 常见动作

- 查看设备最新属性与遥测
- 下载设备证书
- 绑定或解绑规则链
- 通过网关设计器维护采集点位
