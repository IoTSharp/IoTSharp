# 规则链

规则链承担 IoTSharp 的实时数据处理、告警触发和短动作分发能力。

## 菜单入口

- 数字孪生 > 规则链设计
- 数字孪生 > 规则链审计

## 规则链相关页面

当前前端已经包含：

- 规则链列表 `flowlist.vue`
- 规则链设计器 `flowdesigner.vue`
- 规则链模拟器 `flowsimulator.vue`
- 规则链审计 `flowevents.vue`

## 规则链适合做什么

- 数据过滤和清洗
- 条件判断
- 告警触发
- 属性写入
- 遥测转发
- 调用外部接口

规则链不承载长周期发布、OTA、灰度、回滚或审批流程；这些能力归入后续 Release Center。原独立“场景”入口不再作为核心领域维护，业务可视化能力后续归入 Asset View。

## 设计器使用建议

1. 先在列表页创建规则链。
2. 进入设计器配置节点与连线。
3. 使用模拟器验证输入输出。
4. 发布后到审计页观察实际执行记录。
5. 最后把规则链绑定到设备或产品。
