# 规则链与场景

规则链承担了 IoTSharp 的数据处理和业务编排能力。

## 菜单入口

- 数字孪生 > 规则链设计
- 数字孪生 > 规则链审计
- 数字孪生 > 场景

## 规则链相关页面

当前前端已经包含：

- 规则链列表 `flowlist.vue`
- 规则链设计器 `flowdesigner.vue`
- 规则链模拟器 `flowsimulator.vue`
- 规则链审计 `flowevents.vue`

## 规则链适合做什么

- 数据过滤和清洗
- 条件判断
- 告警触发
- 属性写入
- 遥测转发
- 调用外部接口

## 设计器使用建议

1. 先在列表页创建规则链。
2. 进入设计器配置节点与连线。
3. 使用模拟器验证输入输出。
4. 发布后到审计页观察实际执行记录。
5. 最后把规则链绑定到设备或产品。

## 场景页的定位

“场景”适合承载更高层级的业务编排入口，通常可与规则链、资产或产品模型联动使用。
