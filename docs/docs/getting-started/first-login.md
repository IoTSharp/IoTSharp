# 首次登录

初始化完成后，系统进入正常登录流程。

## 登录页特点

当前登录页包含：

- 用户名和密码登录
- 拖拽验证码
- 登录成功后的菜单与权限加载

## 登录后的默认去向

登录成功后通常进入：

- `/dashboard`

系统随后会从 `/api/Menu/GetProfile` 拉取当前用户可见菜单。

## 权限与菜单

不同角色看到的菜单不同：

- 普通用户主要使用仪表盘、数字孪生、产品管理、资产管理。
- 管理员会额外看到系统管理中的租户、客户、用户或证书管理。

## 建议的首次检查项

第一次登录后建议依次确认：

1. 仪表盘是否正常加载。
2. 设备管理列表是否能打开。
3. 规则链列表是否能进入。
4. 个人中心是否能读取当前账号资料。
